## Process

Repository:

```bash
git clone https://github.com/dmnchzl/jarvis.git
```

Install:

```bash
pnpm install
```

Start:

```bash
pnpm run dev
```

Test:

```bash
pnpm run test
```

Build:

```bash
pnpm run build
```

Preview:

```bash
pnpm run preview
```
